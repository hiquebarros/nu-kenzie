# Nu Kenzie

## 1. Sobre

    O Nu Kenzie é um controlador de finanças pessoais que simula a identidade da marca NuBank.
    Esse projeto foi desenvolvido como entrega da primeira sprint de ReactJS
    na Kenzie Academy Brasil.
